# docker-metube
> Docker for metube.

## get-started
```shell
# 1. clone project
cd ~/github
git clone git@github.com:aric-docker/docker-metube.git

# 2. docker start
cd docker-metube
docker-compose up -d

# 3. goto download dir ~/Downloads/metube-downloads
# change icon

# 4. download video
# 4.1. open browser and goto `http://localhost:8888`
```
